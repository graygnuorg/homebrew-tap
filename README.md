# Graygnuorg Tap

This tap provides formula for [pound](https://github.com/graygnuorg/pound) proxy server.

To install it, type:

```sh
brew install graygnuorg/tap/pound
```

Or `brew tap graygnuorg/tap` and then `brew install pound`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "graygnuorg/tap"
brew "pound"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
